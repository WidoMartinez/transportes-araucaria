# Checklist: Optimización de Modales Admin

Antes de dar por finalizada la optimización de un modal, verifica:

- [ ] **Componentización**: ¿Está el modal en su propio archivo fuera de `AdminReservas.jsx`?
- [ ] **Rendimiento**: ¿Se carga el modal de forma diferida o eficiente?
- [ ] **Accesibilidad**: ¿Tiene todos los labels y títulos necesarios?
- [ ] **Mobile**: ¿El modal se ve bien en una pantalla de 375px?
- [ ] **Feedback**: ¿Muestra un spinner en el botón al guardar?
- [ ] **Toasts**: ¿Usa `sonner` / `toast` para éxitos y errores?
- [ ] **Micro-animaciones**: ¿Tiene transiciones suaves al abrirse/cerrarse?
- [ ] **Validación**: ¿Valida los datos antes de enviarlos al backend (client-side validation)?

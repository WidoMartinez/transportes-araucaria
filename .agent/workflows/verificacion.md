# Verificación de Código

Pasos para asegurar que el código cumple con los estándares y funciona correctamente.

1. Ejecutar Linter de JavaScript/React
   ```bash
   npm run lint
   ```

2. (Opcional) Ejecutar pruebas básicas de sistema PHP
   Si has modificado archivos backend PHP, corre este script:
   ```bash
   php test_sistema.php
   ```

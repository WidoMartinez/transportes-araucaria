# Recursos estáticos

Este directorio contiene las imágenes, íconos y otros recursos de la app.

## Archivos requeridos por Expo

Antes de publicar la app, debes agregar:

- `icon.png` — Ícono de la app (1024×1024 px)
- `splash.png` — Pantalla de carga (recomendado 1242×2688 px)
- `adaptive-icon.png` — Ícono adaptativo Android (1024×1024 px)
- `favicon.png` — Favicon para versión web (32×32 px)

## Formatos recomendados

- PNG sin transparencia para `splash.png`
- PNG con transparencia para `icon.png` y `adaptive-icon.png`

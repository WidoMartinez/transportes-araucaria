# Pantallas adicionales

Este directorio contiene pantallas de la app que se agregan como módulos independientes,
fuera del sistema de enrutamiento de Expo Router.

Ejemplos de pantallas que se pueden agregar aquí:
- `PantallaLogin.jsx` — Inicio de sesión
- `PantallaRegistro.jsx` — Registro de usuario
- `PantallaEvaluacion.jsx` — Evaluación post-viaje
